FLEX

  • Para usar flex, primero se debe colocar "display: flex" en el elemento padre
  • Por defecto, la propiedad "flex-direction: row" (Los elementos se alinearan en fila)
  • Por defecto, la propiedad "flex-wrap: nowrap" (Solo trabajara con una fila)
  • Cuando se una propiedad "row reverse" o "column reverse", las propiedades se invierten
  • Cuando se usa propiedad "column", justify-content cambia a vertical y align-items a horizontal.
  • "flex-flow", es una propiedad que combina flex-direction y flex-wrap. Acepta un valor de cada una separada por un espacio.
    Ejem: flex-flow: column-reverse wrap-reverse;
  • "align-content", propiedad que mueve todos los elementos que estan organizados en mas de una fila.
    • Es una propiedad que trabaja obligatoriamente con "flex-wrap: wrap"
    • Sus valores son: flex-start, flex-end, center, space-between, space-around y stretch.

"flex wrap: wrap"

  • Por defecto, al usar flex la propiedad es "flex-wrap: nowrap" (Solo trabajara con una fila)
  • "flex-wrap:wrap", permite crear mas filas cuando se usa flex
  • Existe la propiedad "flex-wrap:wrap-reverse" , permite crear mas filas cuando se usa flex pero invierte el orden de los elementos.
  • Para determinar la cantidad de items, se dividira el 100% (Ancho total de la pantalla) entre la cantidad de items a colocar.
    Ejemplo: 100% / 4 items = 25%
    Este resultado debe ir en la propiedad "width" del contenedor de los items. (width: 25%;)
