Imagenes

1. Convertir imagen cuadrada a redonda

  • Para realiar la conversion, usaremos :"border-radius: 50%" + "object-fit: contain;"
  • Existe el "object-fit: cover", que adapta su contenido pero segun el ancho.

2. Opacidad de una imagen

  • Para opacara una imagen, usaremos :"opacity: 0-100"

3. Filter - filter: grayscale(100%);

  • Para conseguir una imagen en escala de grises, usaremos "grayscale" con valores entre 0 - 100%

4. Filter - filter: blur(5px);

  • Aplica el efecto "blur" con valores del 0 - 100px

5. Filter - filter: sepia(100%);

  • Aplica el efecto "sepia" con valores del 0 - 100%

6. Filter - filter: invert(90%);

  • Aplica el efecto "negativo" con valores del 0 - 100%

7. Filter - filter: hue-rotate(190deg);

  • Aplica el efecto "hue-rotate" con valores del 0 - 360deg

8. Filter - filter: saturate(50%);

  • Aplica el efecto "saturacion" con valores del 0-200%

9. Filter - filter: brightness(0);

  • Aplica el efecto "brillo" con valores del 0-1

10. Filter - filter: opacity(50%);

  • Aplica el efecto "opacidad" con valores del 0-100%
